## 🛠️ 本地开发

如果您想在本地运行这个项目，请按照以下步骤操作：

1. **克隆仓库**
   ```bash
   git clone https://github.com/zhaoyihuan/zhaoyihuan.github.io.git
   cd zhaoyihuan.github.io
   ```

2. **打开网站**
   - 直接双击 `index.html` 文件
   - 或者使用本地服务器（推荐）：
     ```bash
     # 使用 Python 3
     python -m http.server 8000
     
     # 使用 Node.js
     npx serve .
     ```

3. **访问网站**
   在浏览器中打开 `http://localhost:8000`
